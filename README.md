# Where Is Daren
This program runs on a small desktop tablet. It informs the user of my location within the building. It has a secret easter egg feature built into it that, when found, starts a
scavenger hunt. If the user is clever enough to solve all the riddles and find where the treasure chest is physically located in the building, and can enter the secret
combination, they are rewarded (they have 'Found' Daren).
